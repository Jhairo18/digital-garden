# Que es
Unidad de medida que indica el rendimiento de un procesador, sus siglas significa Tera Operaciones por Segundo.

Mide cuantos billones (millones de millones) de operaciones matemáticas básicas puede realizar un chip en un solo segundo.
# Medidas de TOPS
- Basico (10-15 Tops): Funciones sencillas de IA
- Avanzada (40+ Tops): Necesario para las funciones nuevas de IA generativa y asistentes inteligentes
- Profesional (100+Tops): Edicion de alta gama, renderizado 3D y desarrollo de software de IA.
